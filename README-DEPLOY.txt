=== Netlify Drop 배포 가이드 ===

1) 브라우저에서 다음 URL 접속:
   https://app.netlify.com/drop

2) 이 폴더 (또는 옆에 있는 market-brief-deploy.zip 파일)를
   페이지 가운데의 점선 박스(Drag & drop your site folder here)
   안으로 드래그앤드롭.

3) 약 10~30초 후 자동으로 URL이 생성됩니다.
   예: https://gleaming-puffpuff-12345.netlify.app

4) URL을 핸드폰/PC 어디서든 접속하면 동일한 대시보드가 보입니다.

5) (선택) URL을 영구히 보존하려면 화면 안내에 따라 무료 가입 후
   "Claim this site" 클릭. 가입하지 않으면 24시간 후 사이트가 삭제됩니다.

6) (선택) 사이트 이름 변경: Netlify 대시보드에서
   Site settings → Change site name → 원하는 이름 입력
   예: damon-market-brief.netlify.app

7) 업데이트가 필요할 때마다 Claude에게 말씀하시면 새 zip을 만들어드립니다.
   같은 사이트에 다시 드래그앤드롭하면 즉시 반영.
